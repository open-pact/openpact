## Looking for an OpenClaw Alternative?

If you've been using OpenClaw and found yourself wishing for something that takes security more seriously, OpenPact might be exactly what you're looking for.

We've built OpenPact from the ground up with **security as a core principle**, not an afterthought. Here's what that means in practice:

- **Sandboxed Script Execution**: Every Starlark script runs in complete isolation. No surprise file system access, no sneaky network calls.
- **Principle of Least Privilege**: Tools and scripts only get access to exactly what they need - nothing more.
- **Proper Secret Management**: Your API keys and credentials are handled securely, with no accidental exposure in logs or outputs.
- **Script Approval Workflows**: New scripts require admin approval before they can run, giving you full control over what your AI can do.
